# 任务计划

## 项目概述
**KuaFu 项目管理平台** — 面向公司开发和项目管理人员的日常项目管理可交互平台，风格简洁专业。

## 目标用户
- 公司开发人员
- 项目管理人员（PM）

## 核心功能范围（待确认）
- [ ] 项目列表与详情
- [ ] 任务看板（Kanban）
- [ ] 任务创建、分配、状态流转
- [ ] 成员管理
- [ ] 进度追踪 / 甘特图或日历视图
- [ ] 通知与评论

## 关键决策
- **设计风格：** Micro-interactions + Minimal Single Column
- **主色：** Professional Blue `#2563EB` + Deal Green `#059669`
- **字体：** Plus Jakarta Sans（标题 + 正文统一）
- **图标：** SVG 矢量图标（Heroicons 或 Lucide），禁止使用 emoji
