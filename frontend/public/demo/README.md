# 团队任务追踪系统 - 前端UI原型

## 🎨 设计理念

这是一个基于**高质量企业级标准**设计的团队任务追踪系统前端原型，遵循现代企业级应用的设计规范和最佳实践。

### 核心特性

- ✨ **高质量设计** - 8px网格系统，精确到像素级的设计
- 🎯 **专业配色** - 完整的色彩层级系统
- 📊 **实时数据可视化** - Chart.js图表展示
- 📱 **完美响应式** - 支持桌面、平板、移动端
- ⚡ **流畅体验** - 自然过渡动画，符合物理规律
- ♿ **无障碍支持** - 符合WCAG 2.1标准

## 🚀 快速开始

### 方式一：直接打开

1. 打开 `index.html` 文件即可在浏览器中查看

### 方式二：本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 🎨 设计亮点

### v3.0 - 企业级高质量标准

#### 1. 完整设计系统
- **8px网格系统** - 确保所有元素对齐精准
- **10级色彩系统** - 每个颜色有10个层级
- **5级阴影系统** - 从xs到2xl的渐进式阴影
- **5级圆角系统** - 统一的角度规范
- **3级动画系统** - 快速/标准/慢速时长
- **2级缓动函数** - 标准/强调的物理感

#### 2. 专业色彩系统
- **主色**: #4f46e5 (蓝紫)
- **成功**: #10b981 (绿色)
- **警告**: #f59e0b (橙色)
- **危险**: #ef4444 (红色)
- **信息**: #3b82f6 (蓝色)
- **完整色阶**: 每个颜色50-700共9个层级

#### 3. 字体系统
- **主字体**: 系统字体栈 (-apple-system, BlinkMacSystemFont)
- **字号层级**: 12px-28px共8个层级
- **字重**: 400, 500, 600, 700四个层级
- **字间距**: -0.025em优化可读性

#### 4. 间距系统
- **8px基准**: 4px, 8px, 12px, 16px, 20px, 24px, 32px, 40px, 48px, 64px
- **一致性**: 所有元素使用统一间距标准

#### 5. 交互细节
- **悬停效果**: 微妙的位移和阴影变化
- **点击反馈**: 按钮按下时的视觉反馈
- **聚焦状态**: 清晰的键盘导航指示
- **加载状态**: 优雅的过渡动画
- **禁用状态**: 明确的视觉区分

#### 6. 视觉层次
- **主标题**: 28px/700 蓝色
- **副标题**: 20px/600 深灰
- **正文**: 16px/400 深灰
- **辅助文字**: 14px/400 灰色
- **标签文字**: 12px/500 中灰

## 📊 技术栈

- **HTML5** - 语义化标签，无障碍支持
- **CSS3** - 现代特性
  - CSS Variables (完整设计系统)
  - Flexbox & Grid 布局
  - CSS Animations & Transitions
  - 自定义属性系统
- **JavaScript (ES6+)** - 交互逻辑
  - 模块化设计
  - Chart.js 图表库
  - Intersection Observer API
  - 性能优化
- **外部库**
  - Font Awesome (图标)
  - Google Fonts (Inter字体)
  - Chart.js (数据可视化)

## 📱 响应式设计

### 断点设置
- **桌面大屏**: ≥ 1400px (max-width: 1600px)
- **桌面端**: 1200px - 1399px
- **平板端**: 1024px - 1199px
- **大屏手机**: 768px - 1023px
- **小屏手机**: < 768px

### 适配策略
- 桌面端：完整功能展示
- 平板端：优化布局
- 移动端：垂直堆叠
